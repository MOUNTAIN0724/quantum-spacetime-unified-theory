# 行為準則

## 🌟 我們的承諾

為了營造一個開放、友好、多元的環境，我們作為貢獻者和維護者承諾：讓參與我們項目和社區的每個人都不受騷擾，無論其年齡、體型、殘疾、民族、性別認同和表達、經驗水平、國籍、個人外表、種族、宗教或性取向如何。

## 🌈 我們的標準

有助於創造積極環境的行為包括：

### 積極行為
- **尊重和友善**：以尊重和友善的態度對待所有人
- **開放包容**：歡迎不同的觀點和經驗
- **建設性反饋**：提供和接受建設性的反饋
- **責任擔當**：為自己的錯誤道歉，並從經驗中學習
- **關注社區**：關注什麼對整個社區最有利

### 不可接受的行為包括：
- **騷擾言論**：性別化語言或意象，不受歡迎的性關注或挑逗
- **攻擊行為**：侮辱性/貶損性評論，人身或政治攻擊
- **公開羞辱**：公開或私下騷擾
- **未經授權**：未經明確許可發布他人的私人信息
- **其他行為**：在專業環境中被合理認為不適當的其他行為

## 🛡️ 維護者責任

項目維護者有責任澄清可接受行為的標準，並對任何不可接受的行為採取適當和公平的糾正措施。

項目維護者有權力和責任刪除、編輯或拒絕不符合本行為準則的評論、提交、代碼、wiki編輯、問題和其他貢獻，或因其他正當原因暫時或永久禁止任何貢獻者。

## 📜 適用範圍

本行為準則適用於所有項目空間，以及個人在公共空間代表項目或其社區時。代表項目或社區的例子包括使用官方項目電子郵件地址、通過官方社交媒體賬戶發布，或在線上或線下活動中擔任指定代表。

## ⚖️ 執法

可以通過以下方式報告虐待、騷擾或其他不可接受的行為：
- **電子郵件**: qst-conduct@example.com
- **GitHub Issues**: 私密報告給維護者

所有投訴將被審查和調查，並將產生被認為必要和適當於情況的回應。項目團隊有義務對事件報告者保密。

## 🔄 執法指導方針

項目維護者將遵循以下社區影響指導方針來確定違規行為的後果：

### 1. 糾正
**社區影響**：使用不適當的語言或其他被社區認為不專業或不受歡迎的行為。

**後果**：由維護者發出非公開的書面警告，澄清違規行為的性質，並解釋為什麼行為不適當。可能會要求公開道歉。

### 2. 警告
**社區影響**：通過單一事件或一系列行動發生的違規行為。

**後果**：警告，指出持續行為的後果。在指定時間內不與涉及人員互動，包括主動與執行行為準則的人員互動。這包括避免在社區空間和外部渠道中的互動。違反這些條款可能會導致臨時或永久禁令。

### 3. 臨時禁令
**社區影響**：嚴重違反社區標準，包括持續不適當行為。

**後果**：在指定時間內禁止任何形式的互動或與社區的公開交流。在此期間不允許與涉及人員公開或私下互動，包括主動與執行行為準則的人員互動。違反這些條款可能會導致永久禁令。

### 4. 永久禁令
**社區影響**：表現出違反社區標準的模式，包括持續不適當行為、騷擾個人、或對個人或群體的攻擊或貶損。

**後果**：永久禁止任何形式的社區內公開互動。

## 📖 歸屬

本行為準則改編自 [Contributor Covenant](https://www.contributor-covenant.org) 2.1版，可在 https://www.contributor-covenant.org/version/2/1/code_of_conduct.html 查看。

社區影響指導方針靈感來自 [Mozilla's code of conduct enforcement ladder](https://github.com/mozilla/diversity)。

## 🤝 我們的期望

### 學術討論
- **尊重不同意見**：理論物理有不同觀點是正常的
- **基於證據**：討論應基於數學推導和觀測證據
- **開放質疑**：歡迎對理論的建設性質疑
- **共同進步**：目標是共同推進對宇宙的理解

### 合作精神
- **相互學習**：每個人都有值得學習的地方
- **分享知識**：樂於分享您的專業知識
- **幫助他人**：幫助新成員融入社區
- **團隊合作**：共同解決複雜問題

### 科學誠信
- **誠實報告**：誠實報告結果，包括負面結果
- **尊重數據**：不篡改或選擇性使用數據
- **承認貢獻**：適當承認他人的貢獻
- **透明過程**：保持研究過程透明

## 💫 我們的願景

我們希望建立一個：
- **開放**：對新想法和不同觀點開放
- **嚴謹**：堅持科學嚴謹性和誠信
- **包容**：歡迎各種背景的貢獻者
- **創新**：鼓勵創新思維和原創研究
- **合作**：促進跨學科合作和交流
