## 🔧 理論修正

### 新參數：環境依賴的a_eff
a_eff(σ) = a₀_standard × f(σ)
f(σ) = f₀ × (σ/σ₀)^α

text

其中：
- σ: 表面密度 (10⁹ M_sun/kpc²)
- f₀ = 1.0 (正常密度時)
- σ₀ = 10.0 (參考密度)
- α = 2.0-3.0 (密度指數)

對於矮星系：
σ < 0.5 → f(σ) ≈ 0.001-0.01

### 物理機制
時空中子密度依賴性：
1. 低密度環境：時空中子間距增大
2. Φ⁺/Φ⁻場不對稱性增強
3. Ω場進入"超有序相"
4. 有效加速度尺度極度縮小

### 修正的尺度依賴函數
β_eff(M, σ) = β₀ × g(M/M_th) × h(σ)

其中：
g(x) = exp(-(1-x)) if x<1 else 1-exp(-(x-1))
h(σ) = (σ/σ_crit)^β, σ_crit = 0.1, β = 0.5
